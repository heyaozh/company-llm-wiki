# Index

The at-a-glance list of every page in the wiki, grouped by **surface → type**. This is metadata,
not a document. It is rebuilt by the `reindex-wiki` skill from page properties + labels.

In Confluence, prefer a **Page Properties Report** macro per section (keyed on the `surface:*`
and `type:*` labels) so the table stays live as pages are added — the macro reads the
`id`, `title`, `status`, `owner`, `version`, `review_year`, `completeness` properties directly.
Use the static tables below only as the seed / fallback layout.

Columns: **id** · **title** · **status** · **owner** · **version** · **review_year** · **completeness**
